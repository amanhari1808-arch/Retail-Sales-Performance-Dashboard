# Retail-Sales-Analytics-Dashboard-India (Interactive Dashboard creation using Power BI)
## Project Objective
The objective of this project is to analyze retail sales data using Power BI to uncover meaningful insights about sales performance, customer behaviour, and business trends. The dashboard focuses on evaluating key performance indicators such as total sales, total profit, total cost, average unit price, and total orders. It provides an interactive platform to explore patterns across product lines, customer segments, cities, payment methods, and time periods.
